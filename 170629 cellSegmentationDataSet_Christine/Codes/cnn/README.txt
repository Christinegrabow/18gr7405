cnn
---

- Training is done by cnn_train_test_valid.lua.
- Command is "th cnn_train_test_valid.lua".
- All the parameters can be set inside the script.
------------------------------------------------------------------

- To predict labels, use predict.lua.
- Command is "th predict.lua".
- It generates 3 maps which are stored in "test_image_dir/tmp".
- The generated maps are rotated by 180 degree.
- Parameters can be set inside the script.
------------------------------------------------------------------
